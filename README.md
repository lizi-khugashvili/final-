# Cosmic Library Storefront

## Project Name
Cosmic Library Storefront

## Short Description
Cosmic Library Storefront is a full-stack online bookstore application.  
The project allows users to browse a collection of books, search by title, filter by genre, sort by price, view detailed information about each book, and manage a shopping cart. Users can also send a checkout request through the backend API.

## Technologies Used

### Frontend
- HTML
- CSS
- Vanilla JavaScript

### Backend
- Node.js
- Express.js
- SQLite

## Features

- Books catalog displayed as styled cards
- Search books by title
- Filter books by genre
- Sort books by price (low to high / high to low)
- Book details modal window
- Cart drawer (add, remove, update quantity)
- Checkout modal with user input (name, email, phone)
- Toast notifications (add to cart, remove item, checkout)
- Loading skeletons while fetching data
- Empty state UI when no results are found
- Responsive design (desktop, tablet, mobile)
- LocalStorage support for cart persistence
- Backend API integration
- SQLite database for storing data
